# Pam - Organization Agent (Standards-aligned)

This folder defines Pam's standards-governed contracts for organization workflows across two owned data surfaces:

- `pam_reading_items` (Reading DB)
- `pam_todos` (Todo DB)

Capabilities covered in this version:

- Reading capture, categorization, unread queue listing, read transition, and search/filter.
- Todo capture, lifecycle updates (complete/reopen/defer), and prioritization.
- Focus queue and concise reminder-nudge generation.
