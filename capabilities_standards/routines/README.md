# Routines Standards Index

This directory is split into two tracks:

- `versioning/` — routine JSON versioning and active-pointer process.
- `migrations/` — reusable agent migration playbooks, templates, and examples.

## Start here
1. If you are publishing/updating workout routines, use `versioning/README.md`.
2. If you are migrating an agent to standards, use `migrations/README.md`.
